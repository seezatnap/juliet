# Specifications: sonnets

# Sonnets Collection

## Goal
Create a `sonnets/` directory in the project root and populate it with ten original sonnets.

## Requirements
- Create a `sonnets/` directory at the project root.
- Write ten sonnet files inside `sonnets/`, named `sonnet-01.md` through `sonnet-10.md`.
- Each file must contain a single original sonnet in traditional 14-line form (three quatrains and a couplet, or other recognized sonnet structures).
- Each sonnet should have a title as a markdown heading.
- Sonnets should cover a variety of themes (nature, love, time, art, etc.).
