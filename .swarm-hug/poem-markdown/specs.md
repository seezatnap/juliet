# PRD: Poem Markdown

## Overview
Create a single Markdown file containing an original poem about Harry Potter. The output should be readable and nicely formatted in Markdown.

## Goals
- Produce one Markdown file with a poem.
- Keep formatting simple and compatible with standard Markdown renderers.

## Requirements
- The file must be Markdown (`.md`).
- Include a title and the poem body.
- The poem should be original, suitable for a general audience, and themed around Harry Potter.
- Do not quote or closely paraphrase any copyrighted passages.

## Non-Goals
- No images or external links.
- No additional documentation.

## Acceptance Criteria
- A single `.md` file exists with a poem title and body themed around Harry Potter.
- The poem renders cleanly in Markdown.
